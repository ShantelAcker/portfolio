Projects

Weather App

A weather app made with ReactJS, OpenWeatherMap API, and Axios. The user can enter a zip code and the current weather information for that zip code will be displayed, along with an icon that illustrates the weather conditions.

Password Generator

A password generator made with HTML, CSS, and Vanilla JavaScript. The user can choose between a few different parameters to customize the type of password generated, and there is a copy button to conveniently use the generated password.

JavaScript Clock

A clock app made with HTML, CSS, and Vanilla JavaScript. The data and time will automatically be updated based on the user's locale, and the clock is dark-mode only so the user can have clock that is easy on the eyes.
